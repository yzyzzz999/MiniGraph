# MiniGraph 评测数据集

## 评测目标
对比不同方案的效果：
1. 纯 LLM（无 RAG）
2. RAG（单跳）
3. RAG + 多跳推理
4. RAG + 关系链增强

## 评测维度
- 准确率（Accuracy）
- 召回率（Recall）
- 幻觉率（Hallucination Rate）
- 平均响应时间

## 评测指标定义

### 准确率（Accuracy）
- 完全匹配：预测答案包含标准答案关键词
- 部分匹配：语义相关但不完全准确（人工判断）

### 召回率（Recall）
- 对于多答案问题，预测答案覆盖标准答案的比例

### 幻觉率（Hallucination Rate）
- 预测答案中包含编造信息的比例
- 通过人工检查或关键词匹配判断

### 平均响应时间
- 从请求发送到收到响应的时间（秒）
